### شماره تسک در جیرا:

### توضیحات:

این کامپوننت یک اسلایدر است که می توان توسط آن یک بازه یا یک مقدار عددی را انتخاب کرد.

دو حالت کلی default و range  دارد که در حالت default یک مقدار و در حالت range یک بازه از کاربر گرفته می شود.


### props ها و قابلیت ها:

- **color** دو تم primary و error
- **centerLabel** اگر true باشد لیبل ها به صورت وسط در وسط با خط بالایی خود قرار می گیرند.
- **range** اگر true باشد بازه از کاربر دریافت می شود.
- **verified** اگر true باشد آیکون تیک و اگر false باشد آیکون warning و اگر مقدار نداشته باشد چیزی نشان داده نمی شود.
- **onChange**
- **value** اگر range باشد ، آنگاه value به صورت [number , number ] است و در غیر اینصورت به صورت number.
- **defaultValue**  مثل value
- **marks** یک object که key های آن نشانگر های آبی رنگ هستند و value های آن می توانند object ای از اطلاعات آن نقطه باشند یا خود label باشند. اگر object باشد آنگاه step را نیز برای هر بازه ی کوچک تر داریم و تا نقطه ی بعد باید از همان step استفاده کنیم.
- **min**
- **max**
- **step** شمارنده های بین هر دو نقطه به طور default
