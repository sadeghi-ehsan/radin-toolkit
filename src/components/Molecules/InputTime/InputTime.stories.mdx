### توضیحات :

کامپوننت تایم به طور مستقیم از کاپوننت های FieldGroup و InputNumber و همین طور switch استفاده میکند. استایل های رپر این کامپوننت می تواند از استایل های اینپوت اصلی آن جدا باشد و در حالی که استایل رپر outline یا fill است، آنگاه استایل اینپوت چیز دیگری باشد.

دارای فیچر های کیبورد می باشد و بخشی از آن به خاطر استفاده از کامپوننت InputNumber است.


### props ها و قابلیت ها:


- **groupLabel** لیبل برای فیلدگروپ
- **onChange**
- **value** مقدار باید به صورت ۱۰:۴۵:۳۰ باشد. همین طور مقدار بازگردانده شده نیز به همین صورت است.
- **fieldsetClassName** کلاس برای فیلدست فیلدگروپ
- **fieldGroupVariant** استایل برای اعمال بر رپر اینپوت ها یا همان فیلدگروپ آنها
- **inputVariant** استایل برای اعمال بر اینپوت ها
- **timeFields**یک array به شکل `[{groupLabel:"H",key:"h"},{groupLabel:"M",key:"m"}]` که حاوی آبجکت های از اطلاعات اینپوت های اصلی کامپوننت است. نشان می دهد که آیا چه مشخصاتی از زمان قرار است گرفته شود. key می تواند مقادیر زیر باشد.:
    - h
    - m
    - s
    - ms

- **switchClasses** یک آبجکت به شکل {outerClassName, innerClassName, labelClassName } که حاوی کلاس های کامپوننت switch است.
    - outerClassName : کلاسی برای قسمت آبی یا رپر تمام باتن ها
    - innerClassName : کلاسی برای قسمت متحرک کامپوننت سوییچ که رنگ سفید دارد 
    - labelClassName : کلاسی برای لیبل یا تکست هر باتن که متصل به یک اینپوت است.
