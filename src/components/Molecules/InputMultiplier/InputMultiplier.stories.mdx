### توضیحات:
کامپوننت InputMultiplier به طور مستقیم از کامپوننت input ارث بری میکند. جهت مشاهده کامپوننت input میتوانید به داکیومنت آن مراجعه کنید. با این تفاوت که این کامپوننت دو ورودی و دو خروجی  دارد. این دو ورودی یکی برای input میباشد که کاربر تایپ میکند و دیگری برای مقداری است که در دایره های بالای اینپوت کاربر انتخاب میکند.


فرمت value  و همچنین مقداری که در onchange برگردانده میشود به شرح زیر است:

```json
 {
        "value": 10000,
        "selectedOption": "3x"
 }
```

### props ها و قابلیت ها:

- **options** آپشن هایی که در دایره های بالایی نمایش داده میشوند. این پراپ اجباریست و فرمت آن به شرح زیر است:
```json
 [
    {
        "label": "3x",
        "value": "3"
    },
    {
        "label": "2X",
        "value": "2"
    }
]
```

- **firstOptionAsDefault** تعیین میکند اولین گزینه به عنوان گزینه ی پیش فرض در حالت انتخاب قرار بگیرد یا نه
