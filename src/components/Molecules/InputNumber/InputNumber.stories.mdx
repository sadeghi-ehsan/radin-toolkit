### شماره تسک در جیرا:

### توضیحات:

این کامپوننت همان اینپوت عددی است که به طور مستقیم از کامپوننت Input استفاده میکند. یک سری قابلیت های عوض کردن مقدار آن با دکمه های بالا و پایین کیبورد و همین طور اسکرول هم وجود دارد. 

### props ها و قابلیت ها:

- پراپ های Input را میتواند به آن داد. (به استثنای children و type )
- **baseInputClassName** کلاسی که به baseInput می رسد.
- **value** مقدار ورودی کامپوننت که از جنس number میباشد
- **counterType** مدل باتن های اضافه و کم کردن
- **min** کمترین مقدار مجاز
- **max** بیشترین مقدار مجاز
- **negativesInRed** برای نشان دادن اعداد منفی به رنگ قرمز
- **positivesInGreen** برای نشان دادن اعداد مثبت به رنگ سبز
- **commaSeprated** درصورتی که برابر با true باشد مقدار کامپوننت به صورت comma comma seprated نمایش داده میشود
